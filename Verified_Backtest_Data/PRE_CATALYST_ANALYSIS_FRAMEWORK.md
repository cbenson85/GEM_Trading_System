## üìê QUANTIFICATION METHODOLOGY

### **How to Measure Each Data Point**

**1. Binary Flags (Yes/No)**
- Pattern present or not
- Example: "Consolidation before breakout" = TRUE/FALSE
- Example: "Insider cluster buying" = TRUE/FALSE
- Example: "Golden cross" = TRUE/FALSE

**2. Percentage Metrics**
- Normalize to comparable scale
- Example: "Volume spike = 350% of average"
- Example: "Short interest = 25% of float"
- Example: "Price consolidated within 15% range"
- Example: "Institutional ownership increased 12%"

**3. Count Metrics**
- Simple frequency counts
- Example: "45 days of consolidation"
- Example: "3 insider purchases"
- Example: "7 news articles"
- Example: "4 analyst upgrades"

**4. Ratio Metrics**
- Comparative measurements
- Example: "Put/Call ratio = 0.65"
- Example: "Insider buy/sell ratio = 8:1"
- Example: "News positive/negative ratio = 4:1"

**5. Score/Index Metrics**
- Composite measurements
- Example: "Glassdoor CEO rating = 87/100"
- Example: "Sentiment score = +0.72 (-1 to +1 scale)"
- Example: "Relative strength = 95 (0-100 percentile)"

**6. Temporal Metrics**
- Time-based measurements
- Example: "Days to catalyst = 18"
- Example: "Days since last insider buy = 5"
- Example: "Consolidation duration = 42 days"

**7. Dollar Value Metrics**
- Financial measurements
- Example: "Insider purchases = $2.5M"
- Example: "Options flow = +$4.2M bullish"
- Example: "Market cap = $85M"

## üéØ SUCCESS METRICS FOR PATTERNS

### **What Makes a "Predictive" Pattern?**

**1. High Prevalence (>50% occurrence)**
- Appears in majority of explosive stocks
- Consistently present across years
- Works in different sectors

**2. Strong Correlation (>0.60)**
- Statistical significance
- Consistent relationship with outcome
- Not random chance

**3. Early Signal (Lead Time)**
- Appears BEFORE explosion
- Provides actionable warning
- Enough time to enter position
- Ideal: 7-30 days lead time

**4. Measurable & Screenable**
- Can quantify objectively
- Can screen programmatically
- Available in real-time
- API or scraper accessible

**5. Low False Positive Rate**
- Doesn't appear in non-explosive stocks
- Specific to winners
- Filters out noise

**6. Actionable**
- Clear entry signal
- Defined risk parameters
- Repeatable process
